

### 位运算符



| 含义     | 运算符 | 示例                                                         |
| -------- | ------ | ------------------------------------------------------------ |
| 左移     | <<     | 3 << 1 = 6  ===> 0011 << 1   ==== 0110 （3 * 2^n） n 是移动几位 |
| 右移     | >>     | 同上 除以 2^n                                                |
|          |        |                                                              |
| 按位或   | \|     | 相同位只要一个为1即为1，否则为0。常用于指定位的无条件赋值    |
| 按位与   | &      | 相同位的两个数字都为1，则为1，否则为0。常用于检查指定位是否置位； |
| 按位取反 | ~      | 将0和1全部取反；                                             |
| 按位异或 | ^      | 相同位不同则为1，否则为0。常用于位值翻转                     |
